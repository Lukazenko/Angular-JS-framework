Lihtne hinna kalkulaator. Kui on vaja, et Angularjs midagi kokku arvutaks siis tuleb lisada { {a + b} } sulud ümber tehte.

Kood

<div data-ng-app="" data-ng-init="kogus=1;hind=5">
  <p>Hinna kalkulaator</p>
  Kogus: <input type="number" ng-model="kogus">
  Hind: <input type="number" ng-model="hind">
  <p>Kokku: { { kogus * hind } } </p>
</div>
