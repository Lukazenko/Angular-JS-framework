AngularJS võimaldab teil HTML-i laiendada uute atribuutidega, mida nimetatakse direktiivideks. Direktiivid algavad ng- eesliidesega. Vajadusel saab neid ka ise juurde luua. Antud näidetes kasutan ng-app, ng-init,ng-repeat ja ng-model direktiive.

  • ng-app- Defines the root element of an application.
  • ng-init - Defines initial values for an application.
  • ng-model - Binds the value of HTML controls to application data.
  • ng-repeat - Defines a template for each data in a collection.
